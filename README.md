# Turtle-Keyboard-Drawing

# 🐢 Turtle Keyboard Drawing App

A simple interactive drawing application built using Python's Turtle module.  
Control the turtle using your keyboard to draw freely on the screen.

---

## 🎮 Controls

| Key | Action |
|-----|--------|
| W   | Move forward |
| S   | Move backward |
| A   | Turn left |
| D   | Turn right |
| C   | Clear screen & reset |

---

## 🚀 Features

- Real-time keyboard control
- Smooth turtle movement
- Reset and clear functionality
- Beginner-friendly Python project

---

## 🛠️ Tech Used

- Python
- Turtle Graphics Module
